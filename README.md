A very small bootloader

## Prerequesites
- i8086/i8088 or better CPU
- 640k RAM or more
- A hard drive

## Prerequisites (building)
- NASM assembler

### Installing prerequisites (Debian)
```sh
$ sudo apt-get install nasm
```

### Installing prerequisites (Arch)
```sh
$ sudo pacman -S nasm
```
